## 模板字符串
es6对字符串的扩展很多 对`Unicode`表示法更加完善 
最常用就是`模板字符串`
```
let name = 'lee'
console.log(`hello ${name}`) // 'hello' lee
```
这太好用了 之前我们要用字符串模板 都是拼接
```
`hello` + name + 'good' 
```
这样繁琐也不好看
要注意的是 我们这用反引号`` ` `` 来加强字符串模板
要使用`` ` ``反引号 要加入反斜杠进行转义
```
`hello \`haha\`` // 
```
