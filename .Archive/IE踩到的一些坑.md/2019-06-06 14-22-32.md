# 记录开发过程中遇到的IE坑

## event.path的一些小问题
1. `event.path`很方便的获取所有路径节点。但是在`IE11`、`firefox`中就会收到错误`event.path undefined`.
解决办法:
```
function composedPath (el) {
    var path = [];
    while (el) {
        path.push(el);
        if (el.tagName === 'HTML') {
            path.push(document);
            path.push(window);
            return path;
       }
       el = el.parentElement;
    }
}
var path = event.path || (event.composedPath && event.composedPath()) || composedPath(event.target);
```
